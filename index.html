SemWebCentral

This is the relocation of the Web site SemWebCentral.org, which was originally established as the hosting site for the DARPA Agent Markup Language (DAML) program. Today, SemWebCentral is a GitHub organization located here:

https://github.com/SemWebCentral

SemWebCentral is the parent organization for the open source project Parliament™:

https://github.com/SemWebCentral/parliament

Parliament™ is a semantic graph store, a.k.a., triple store. It's maintenance is sponsored primarily by Raytheon BBN Technologies.
